# Logística - Proveedores y Productos Suministrados

Este proyecto es un módulo web para la gestión de proveedores y productos, desarrollado con Spring Boot, Thymeleaf y PostgreSQL.

## Requerimientos Funcionales
- **RF4.1 - RF4.4**: Gestión de Proveedores (Registrar, Listar, Editar, Inactivar).
- **RF4.5 - RF4.8**: Gestión de Productos (Registrar, Listar, Editar, Inactivar).
- **RF4.9**: Búsqueda de proveedores.
- **RF4.10**: Filtrado de productos.

## Tecnologías
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- Thymeleaf
- PostgreSQL
- Bootstrap 5

## Configuración
1. Crear base de datos PostgreSQL llamada `logistica`.
2. Configurar credenciales en `src/main/resources/application.properties`.
3. Ejecutar con `mvn spring-boot:run`.
